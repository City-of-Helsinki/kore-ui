Kore - Koulurekisteri
=====================

Requirements
------------

- [node.js](http://nodejs.org/)

Development
-----------

Follow the instructions below to set up the development environment.

1. Install npm dependencies:

    ```
    $ npm install
    ```

2. Then, start the development server:

    ```
    $ npm start
    ```

Testing
-------

- Running unit tests:

    ```
    $ npm test
    ```
